KitHack v1

Многофункциональная прошивка для ESP32-WROOM-32

Скачать прошивку

Версия 1.0 - Последнее обновление: 15.06.2025

Скачать KitHack_v1.bin (2.1 MB)

Прошивка совместима с ESP32-WROOM-32 и ESP32-WROOM-32D/E

Функции прошивки

WiFi Функции

  • WiFi Spam - создание множества фейковых точек доступа
  • Evil Portal - фишинговая страница для сбора учетных данных
  • WiFi Jammer - глушение WiFi сигналов
  • Управление WiFi (включение/выключение)

IR Функции

  • IR-B-Gone - выключение телевизоров популярных брендов
  • Project Remote - управление проекторами (Epson, BenQ, Acer)
  • Поддержка NEC протокола
  • Библиотека для сохранения и воспроизведения сигналов

Системные функции

  • Часы с автономным временем
  • SD Files - доступ к файлам на SD карте
  • WebUI - загрузка файлов на устройство через браузер
  • FILES OS - файловый менеджер для LittleFS
  • Настройка времени
