KitHack v1

Многофункциональная прошивка для ESP32 с продвинутыми инструментами

Прошивка KitHack v1

Последняя версия: 1.0 | Дата релиза: 15.06.2025

Скачать прошивку

Размер файла: 1.2 MB | Для ESP32-WROOM-32

WiFi функции

  • WiFi Spam - создает 15 фальшивых точек доступа
  • Evil Portal - фишинговый портал для сбора учетных данных
  • Jammer - глушение WiFi сигналов (2.4 ГГц)

IR функции

  • IR-B-Gone - выключает телевизоры популярных брендов
  • Project Remote - управление проекторами (Epson, BenQ, Acer)

Системные функции

  • Clock - точное время с обновлением на дисплее
  • SD Files - доступ к файлам на SD карте
  • FILES OS - файловый менеджер для LittleFS

Другие функции

  • WebUI - веб-интерфейс для управления устройством
  • NRF24-Hack - инструменты для работы с NRF24
  • CC1101 - сканирование и копирование сигналов
  • Bluetooth Spam/Jammer - спам и глушение BT

Схема подключения

OLED Display (128x64)

SDA  -> GPIO 21
SCL  -> GPIO 22

Кнопки

UP     -> GPIO 32
DOWN   -> GPIO 33
LEFT   -> GPIO 25
RIGHT  -> GPIO 26
SELECT -> GPIO 27
BACK   -> GPIO 34

ИК светодиод

DATA -> GPIO 23

NRF24L01

CE   -> GPIO 2
CSN  -> GPIO 15
MOSI -> GPIO 13
MISO -> GPIO 12
SCK  -> GPIO 14
IRQ  -> Не подключен

SD Card Module

CS   -> GPIO 5
MOSI -> GPIO 23
MISO -> GPIO 19
SCK  -> GPIO 18

CC1101 Module

CS   -> GPIO 4
MOSI -> GPIO 13
MISO -> GPIO 12
SCK  -> GPIO 14
GDO0 -> GPIO 16
GDO2 -> GPIO 35
